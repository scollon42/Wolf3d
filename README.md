# Wolf3d
42 wolf3d project - Raycasting

> 3D program with Raycasting algorithm

### How to use

`Usage : ./wolf3d [-m file.wf]`

**Commands :**

* Use `w` `a` `s` `d` or arrow keys to move
* Use `space` to shoot
* Use `r` to reload
* Use `t` set textures on
* Use `ESC` to quit
